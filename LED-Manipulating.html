LED Manipulating

Using button to control LED flashing

This is a small interactive system based on Arduino coding. Make the EPS32 board connect the LED bulb to a Button through the bread board and set the coding language on Arduino. If people press Botton, the bulb will generate a current loop and the bulb will light up.
At the same time, I set a delay function, if people press and then hold the bottom, the led can blink at a constant speed.

Arduino Coding

In this coding, I wrote two PinMode modes, the first is LED Bulb as Input, and the second is Button as Output. In the if statement, I set if the button's digitalread is active, the LED will flash alternately with high-frequency brightness and low-frequency brightness.
